# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

- `ResourcefulJButton` for localizing button text, icons, mnemonics, and
  accessibility attributes
- `ResourcefulJLabel` for localizing label text, icons, and mnemonics
- `ResourcefulJMenu` for localizing menu text and mnemonics
- `ResourcefulJMenuBar` for localizing menu bar properties
- `ResourcefulJMenuItem` for localizing menu item text and mnemonics
- `ResourcefulJPanel` for localizing panel properties
- `ResourcefulJComboBox` for localizing combo box properties
- `LocaleJMenuItem` for selecting an application locale from a menu
- `LookAndFeelJMenuItem` for selecting a Swing Look and Feel from a menu
- `JComponentPropertyBundle` for defining name, tooltip, font, and
  accessibility properties in resource bundles
- `AbstractButtonPropertyBundle` for defining text, label, mnemonic, and
  icon properties in resource bundles
- `SwingResourcefulDelegate` for centralizing Swing-specific locale
  orchestration and EDT dispatch
- Classpath and module-path test profiles
- Release profile with javadoc and source jar generation
